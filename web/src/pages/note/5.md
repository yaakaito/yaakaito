`--install-extension` というコマンドがあり、これを使うと CLI から拡張機能をインストールできる。
プロジェクトに直接関係しないけど devcontainer に入っていてほしい、みたいなものに個人の設定で使うと便利。

```
code --install-extension mhutchie.git-graph
code --install-extension eamodio.gitlens
```
