TypeScript 5.6 から `--noCheck` オプションが追加され、これまで必ず行われていた型チェックが tsc 単体でもスキップできるようになりました。

```
$ tsc --noCheck
```

swc などが主流で、Node も単体で TS の実行をサポートしようとしていたりする中で、特別用途があるのかというと... という感じです。
