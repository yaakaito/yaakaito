`wrangler dev` コマンドは Cloudflare での動作を想定した dev-server を実行できるコマンドですが、Dev Container 内でこれを実行した場合に発行される URL    にアクセスしても、レスポンスが一切返ってこない状態になりました。コンテナの実行には OrbStack を利用しています。解決法を探してみたところ次のような Issue がありました。

- https://github.com/cloudflare/workers-sdk/issues/6587

どうやら `--ip 127.0.0.1` を加えて起動するとうまくいくことがあるようです。

```shell
$ npx wrangler dev --ip 127.0.0.1
```

今回自分はこれで動作しました。ただし、Dev Container でないローカル環境などではこれでは実行できなくなるので、注意が必要です。
